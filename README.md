# **절대음감 챌린지**

이 소리의 음이름은 무엇일까요? 나의 절대음감 점수를 확인해보세요!

## **플레이 방법**

0. [게임 사이트](https://vrisel.github.io/perfect-pitch-challenge)에 접속한다.
1. Pitch를 선택한다. (기본값 440 ㎐, 범위 416~466)
2. 게임 모드를 골라 시작한다.
   - 일반모드: 단계별로 차례차례 풀기
   - 서바이벌: 세 번 틀리면 끝!
3. 주어지는 소리를 듣고 그 음에 해당하는 건반을 선택한다.
4. 마지막 문제까지 풀면 결과를 SNS에 공유한다.
